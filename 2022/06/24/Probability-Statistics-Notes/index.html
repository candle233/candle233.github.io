# 一 事件与概率

# 1.1 随机试验和随机事件

  1. 随机试验：随机现象的实现和对它某个特征的观测。
  2. 基本事件：随机试验中的每个单一结果，犹如分子中的原子，在化学反应中不可再分。
    e.g. 硬币抛 3 次，有 8 种结果：正正正、正正反、正反正…… 这 8 种可能结果的每一个都是基本事件。
  3. 随机事件：简称事件，在随机试验中我们所关心的可能出现的各种结果，它由一个或若干个基本事件组成。通常用英文大写字母表示或 {一种叙述} 来表示。
  4. 样本空间：随机试验中所有基本事件所构成的集合，通常用Ω\OmegaΩ 或SSS 表示。
    e.g. 掷一枚骰子，观察出现的点数，则Ω={1,2,3,4,5,6}\Omega=\{1,2,3,4,5,6\}Ω={1,2,3,4,5,6}.
  5. 必然事件（Ω\OmegaΩ）：在试验中一定会发生的事件。
  6. 不可能事件（ϕ\phiϕ）：在试验中不可能发生的事件。

# 1.2 事件的运算

  1. 子事件 A⊂BA\subset BA⊂B：事件AAA 发生蕴含时间BBB 一定发生，则时间AAA 成为事件BBB 的子事件。若A⊂BA\subset BA⊂B，且B⊂AB\subset AB⊂A，则称时间AAA 与事件BBB 相等，记为A=BA=BA=B.
  2. 事件的和（A∪BA\cup BA∪B）：事件AAA 和事件BBB 中至少有一个发生称为事件AAA 和事件BBB 的和。
  3. 事件的积（A∩BA\cap BA∩B）：事件AAA 和事件BBB 同时发生称为AAA 和事件BBB 的积。如果A∩B=ϕA\cap B=\phiA∩B=ϕ，则称AAA 和BBB 不相容，即事件AAA 和BBB 不能同时发生。
  4. 对立事件AcA^cAc（或A‾\overline{A}A）：AAA 不发生这一事件称为事件AAA 的对立事件（或余事件）。
  5. 事件AAA 和事件BBB 的差（A−BA-BA−B）：事件AAA 发生而事件BBB 不发生这一事件称为事件AAA 和事件BBB 的差，或等价于ABcAB^cABc.
  6. De Morgan 対偶法则及其推广

A∪B‾=A‾∩B‾,\overline{A\cup B}=\overline{A}\cap \overline{B},A∪B=A∩B,

A∩B‾=A‾∪B‾\overline{A\cap B}=\overline{A}\cup \overline{B}A∩B=A∪B

# 1.3 概率的定义

概率是随机事件发生可能性大小的数字表征，其值在 0 和 1 之间，即概率是事件的函数。概率有以下定义：

# 1.3.1 古典概率

设一个试验有 N 个等可能的结果，而事件EEE 恰包含其中的MMM 个结果，则事件EEE 的概率，记为P(E)P(E)P(E)，定义为

P(E)=M/NP(E)=M/NP(E)=M/N

或

P(E)=#(M)/#(N),P(E)=\#(M) / \#(N),P(E)=#(M)/#(N),

其中，#(M)\#(M)#(M) 为事件MMM 中基本事件的个数。

古典概型有两个条件：

  • 有限性，试验结果只有有限个（记为 n），
  • 等可能性，每个基本时间发生的可能性相同。

注：古典概率可引申出 “几何概率”。

# 1.3.2 概率的统计定义

古典概率的两个条件往往不能满足，但可以将事件的随机试验独立反复做 n 次（Bernouli 试验），设事件AAA 发生了nAn_AnA​ 次，称比值nAn\frac{n_A}{n}nnA​​ 为事件AAA 发生的频率，当 n 越来越大时，频率会在某个值 p 附近波动，且波动越来越小，这个值 p 就定义为事件AAA 的概率。该学派为频率派。

注：不能写为limn→∞nAn=plim_{n\rightarrow{\infty}}\frac{n_A}{n}=plimn→∞​nnA​​=p，因为nAn\frac{n_A}{n}nnA​​ 不是 n 的函数。

# 1.3.3 主观概率

主观概率可以理解为一种心态或倾向性。究其根由，大抵有二：一是根据其经验和知识，二是根据其利害关系。该学派在金融和管理有大量的应用，这一学派成为 Bayes 学派。

# 1.5 条件概率

条件概率就是知道了一定信息下得到的随机事件的概率。设事件AAA 和BBB 是随机试验Ω\OmegaΩ 中的两个事件，P(B)>0P(B)>0P(B)>0，称

# 1.5.2 乘法定理

# 1.8 事件的独立性

设AAA，BBB 是随机试验中的两个事件，若满足P(AB)=P(A)P(B)P(AB)=P(A)P(B)P(AB)=P(A)P(B)，则称事件AAA 和BBB 相互独立。判断事件的独立，应该是从实际出发，如果能够判断事件BBB 的发生与否对事件AAA 的发生与否不产生影响，则事件AAA，BBB 即为独立。

设A~\widetilde{A}A表示事件AAA 发生和不发生之一，B~\widetilde{B}B表示事件BBB 发生和不发生之一。有独立性的定义可推至P(A~B~)=P(A~)P(B~)P(\widetilde{A}\widetilde{B})=P(\widetilde{A})P(\widetilde{B})P(AB)=P(A)P(B)（一共有四个等式）。可推广至：

P(A~1A~2...A~n)=P(A~1)...P(A~n)P(\widetilde{A}_1\widetilde{A}_2...\widetilde{A}_n)=P(\widetilde{A}_1)...P(\widetilde{A}_n)P(A1​A2​...An​)=P(A1​)...P(An​)
