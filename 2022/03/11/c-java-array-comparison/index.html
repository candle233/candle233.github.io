# 相同点：

  • 数组都是从 0 开始索引

  • 都可以用 for 循环进行遍历

  • 都可以通过 array[i]=j 修改数组中的元素值

  • 都可以定义多维数组

# 不同点

# 声明变量与初始化

# C 语言

int a[]={} 为声明了一个没有进行初始化的数组，它的长度是无限的。
未初始化的情况下，数组内的值并不为空，可以进行 printf 。

#include <stdio.h>
int main(){
  int a[]={},i;
  for(i=0;i<10;i++){
    printf("%d,",a[i]);
  }}
/*我们可以得到：
0,10294192,0,4199349,0,0,0,46,0,0,
--------------------------------
Process exited after 0.01153 seconds with return value 2
*/

继续对数组进行实验：

int a[]={1,2,3};//也是一个无限长的数组，不过前三个元素赋了值
int a[1];a[2]=111;//尽管规定了数组的长度，但是依然可以定义溢出长度的值

# Java

java 语言定义数组有两种方式：

  • 第一种： int[] a 定义未初始化的数组

因为没有初始化，输出数组元素会报错

public class test{
    public static void main(String[] args) {
        int[] a;
        System.out.println(a[0]);
    }
}//报错，未进行初始化

如果为其初始化，也不能够输出溢出数组长度的元素

public class test {
    public static void main(String[] args) {
        int[] a={1};
        System.out.println(a[1]);
    }
}//报错,超出了限制
Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: 1

  at test.main(test.java:4)
  • 第二种： int[] a=new int[n] ，其中 n 为数组的长度（必须声明）
public class test {
    public static void main(String[] args) {
        int[] a=new int[5];
        for(int i=0;i<10;i++){
        System.out.print(a[i]+",");
    }}
}//输出一部分，报错一部分，依然超出了限制
/*
0,0,0,0,0,Exception in thread "main" java.lang.ArrayIndexOutOfBoundsException: 5
  at test.main(test.java:5)
  */

# 总结

  • C 语言定义的数组很像指针，每个元素都有初始值（C 语言自己生成的，生成机制不详），并且长度之外的数组元素依然可以打印和赋值。
  • Java 语言有两种定义方法，在长度范围之内的数组元素初值为 0，并且无法打印长度之外的数组元素。

# Java 如何定义类似于 C 语言的数组？

引入一个问题：

输入 n 个数，返回这 n 个数以及最大值和最小值。

问题：：如何不利用 //输入任意非数字字符以结束输入 ****，而通过回车符结束输入？？？？？

//输入任意非数字字符以结束输入
import java.util.Scanner;
public class maxn {
    public static void main(String[] args) {
        int n,max=0;
        Scanner s=new Scanner(System.in);
        while (s.hasNextInt()){
            n=s.nextInt();
            if (n>max){
                max=n;
            }
        }
        System.out.println(max);
    }
}

虽然完全可以不利用数组就可以求解最大值，但是无法将这 n 个数储存在数组中。

为了储存 n 个元素在数组中，关键问题在于 Java 如何定义像 C 语言一样无限长的数组。

这里我们使用：
ArrayList 类：一个可以动态修改的数组，与普通数组的区别就是它是没有固定大小的限制，我们可以添加或删除元素。

上述题目解法：(输入任意非数字字符以结束输入)

import java.util.ArrayList;//输入任意非数字字符以结束输入
import java.util.Scanner;

public class test1 {
    public static void main(String[] args) {
        int max,min;
        Scanner s=new Scanner(System.in);
        ArrayList<Integer> lists=new ArrayList<Integer>();
        while (s.hasNextInt()){
            lists.add(s.nextInt());
        }
        max=min=lists.get(0);
        for(int i:lists){
            max=(i>max)?i:max;//max = Math.max(max,i);
            min= Math.min(i, min);//min = Math.min(min,i)
            System.out.println(i);
            }
        System.out.println("max:"+max+", min:"+min);
        }
    }

ArrayList 类的定义方式和索引方式均与数组不同，且默认 ArrayList 内的元素为 - 1
