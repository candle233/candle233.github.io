# 第一课　李さんは中国人です

  • 名は名です
    相当于 “是”，“は” 是主语部分，“です” 是谓语部分，助词 “は” 用于提示主题，读作 “わ”
    ・李さんは中国人です。
    ・私は日本人です。

  • 名は名ではありません
    相当于 “不是”
    ー森さんは学生ではありません。
    ーわたしは田中じゃない。（口语化）

  • + か表示疑问，结尾不加 “？”
    ーあなたは小野さんですか。
    ーはい、小野です。

    ー木村さんは中国人ですか。
    ーいいえ、中国人ではありません

  • の连接名词和名词
    前为名词，后为名词属性
    注意：不论名词什么关系，加の、例私の父

  • どうぞよろしくお願いします。
    【寒暄语】请多关照

# 第二课　これは本です

  • これ / それ / あれ　は名です
    これ：距说话人较近的事物
    それ：距听话人较近的事物
    あれ：距两人都较远的事物
    ・これは本です
    ・それはカバンです
    ・あれはテレビです
    ・それは何ですか。那是什么？
    ・あの人は誰ですか。那个人是谁？
    注意：日语问句通常没有问号，か表疑问，助词
  • 名の名 【所属】
    の连接名词与名词表示所属
  • この / その / あの
    用于修饰名词
  • どれ与どの
    どれ与どの是三个以上事物，不确定哪一个时所用的疑问词
  • 方【礼貌语言】
    あの方は田中さんです
  • 询问年龄
    おいくつですか
  • どうぞ
    给对方物品或全对方进餐

# 第三课　ここはデパートです

  • ここ / そこ / あそこ
    ・ここはでぱーとです　这里是百货商店
    ・そこは図書館です　那里是图书馆
    ・あそこは入口です　那里是入口
  • 名は名【场所】です
    表示 “名词” 存在于 “名词【场所】”
    ・食度はデパートの 7 階です。
    ・トイレはここです
    ・小野さんは事務所です
    です可翻译为 “是”，也可以广泛译为 “在”
  • 名はどこですか
    询问场所：
    • トイレはどこですか
      • あちらです
    • あなたのばんはどこですか
      • 私のか b んはここです
  • 名も名です
    表示 “也”
    ・ここは JC 企画のビルです
    ・あそこも JC 企画のビルです
  • 名はいくらですか
    询问价钱
    ・これはいくらですか
    ・その服はいくらですか
  • 省略用语
    これはいくらですか
    あれは（音调升高）
  • こちら / そちら / あちら / どちら
    礼貌说法 |
    ・会社ゃはどちらですか
  • ～ですか【确认】
    重复内容后加か表示理解对方的内容
    それはパソコンです
    パソコンですか 音调向下

# 第四课　部屋に机といすがあります

  • あります / います
    表示事物存在时，谓语是あります。
    表示花草等不具有意志的事物，用います
    • 名【场所】に名【物 / 人】があります / います（有）
      ・部屋に机があります。
      ・ここに本があります。
      ・あそこにだれがいますか。
    • 名【物 / 人】は名【场所】にありす / います（在）
      ・椅子は部屋にあります。
      ・図書館はどこにありますか。
  • と
    名と名【并列】
  • 方位词
    うえ上　そと外　なか中　した下　まえ前　うしろ後ろ　ちかく近く
  • 疑问词 + も + 动（否定）
    ・教室にだれもいません 教室里没有人
    ・冷蔵庫に何もありません 冰箱里没有东西

# 第五课　森さんは７時に起きます

  • 今時分です
    ・今何時ですか。
    ・八時三分です。
  • 一类动词（五段动词），以う段结尾
    向前推一段：うー＞い＋ます
    休むー＞休みます
  • 二类动词（い段动词），以る段结尾、る前假名在い、え段上
    去掉る＋ます
    起きる→起きます
    寝る➞寝ます（满足二类动词的五段动词有 26 个）
  • 三类动词（サ变动词），以する结尾的词
    去向前うー＞い＋ます
    勉強するー＞勉強します
  • 动词四种形式
    动ます / 动ません / 动ました / 动ませんでした
  • 名【时间】に动
    • 表示动作发生的时间时，在具体时间上加 “に”
      ・森さんは 7 時に起きます。
    • “会”“昨日”“明日” 等词后不加に，星期后一般加に，日曜日に也可不加。
      ・私は明日休みます
  • 名【时间】から名【时间】まで动
    ・私は９時から５時まで働きます。
    可单独使用
    ・私は９時から働きます。
  • いつ动ますか
    询问动作或事态进行的时间用 “いつ”，具体时间后加 “に”
    試験はいつ始まりますか。
    询问持续性动作或事态的起点或终点时，用 “いつから”“何曜日まで”
    ・展覧会はいつから始まりますか。
  • は【对比】
    は可以表示对比
    ・小野さんは今日は休みます。
  • ごろ
    表示大致的时间范围

# 第六课　吉田さんは来月中国へ行きます

  • 名【场所】へ动
    使用” 行きます”” 帰ります” 等表示移动的动词时，移动的目的地用助词へ表示，读作えへ行きます
    ・吉田さんは中国へ行きます
  • 名【场所】から动
    使用移动动词时，起点用助词から表示
    ・李さんは来月北京から行きます
  • 名【人】と动
    共同做某事的对象用と表示
    ・小野さんは友達と帰りました
  • 名【交通工具】で动
    交通手段用助词で表示，步行用” 歩いて”
    ・上海まで飛行機で行きます
  • 名【场所】から名【场所】まで动
    表示移动范围
    ・森さんは東京から広島まで新幹線で行きます
  • 助词复合
    ・私の部屋には電話がありません
    ・韓国へは行きました。
    ・中国へは行きませんでした
  • たしか
    表示不完全有把握的记忆
  • 家 与 えち
    一般指建筑物用 “家”，着重指家人用えち，えち本身就表示我家，用 “家” 时，必须说 “わたしの家”

# 第七课　李さんは毎日コーヒーを飲みます

  • 名を动
    动词的对象用助词を表示
    李さんは毎日コーヒーを飲みます。
  • 名【场所】で动
    存在的场所和动作进行的场所，前者用 “に” 后者用 “で”。
    ・李さんは図書館で勉強します。
    ・今日どこで新聞を読みましたか。
  • 名か名
    か表示或者
  • 名をください
    买东西或菜用 “～をください” 形式
    ・この本をください。
  • “何” 的几种发音
    何（なに）＋と / が / を
    何（なに / なん）＋で
    何（なん）＋の / 时刻 / 星期等
  • 寒暄用语
    いらしゃいませ 欢迎光临
    いつれいします　失礼します 告辞
    いってまいります 我走了
    いってらっしゃい 你走好
    おかえりなさい　お帰りなさい 你回来了
    おじゃまします　お邪魔します 打扰了

# 第八课　李さんは日本語で手紙を書きます

  • 名【工具】动
    ・李さんは日本語で手紙を書きます。
    ・手紙を速達で送りました。
    ・なんでうどんを作りますか。
  • 名【人】は名【人】に名【物】をあげます　“给”
    用于 1→2/3，2→3，3→3，不用于 2/3→1【人称】
    ・私は小野さんにお土産をあげます
    ・小野さんは森さんにチョコレートをあげました。
  • 名【人】は名【人】に名【物】をもらいます　“得到、接受”
    ・私は小野さんに辞書をもらいました。
    ・私は長島さんから写真をもらいました。
  • 名【人】に会います　“见”
    李さんは明日長島さんに会います。
  • 【提醒】【已经】【刚刚】【刚才】
    【提醒】よ
    【已经】もう
    【刚刚】さっき
    【刚才】たった

# 第九课　四川料理は辛いです

  • 名は一类形です
    ・四川料理は辛いです。
    ・あのスープは冷たいです。
    ・このお茶は熱いです。
  • 一类形容词做谓语的否定形式
    词尾” い” 变成 “く” 加上 “ないです” 或 “ありません”
  • 一类形做谓语时的过去形式
    词尾 “い” 变 “かった” 加上 “です”
    过去否定形式则为去 “い” 加 “くなかったです” 或 “くありませんでした”。
    ・旅行は楽しかったです。
  • “を”→“は”
    “コーヒーをのみます” 中的 “を” 前名词作话题或进行对比时，用 “は” 代替 “を”
  • あまり与否定形式呼应，表示程度不高
    ちょうどいい表示时间或分量等合适
  • 程度副词
    （很非常）とても / たいへん >（一点儿）少し / ちょっと＞（不太）あまり… ません＞（根本不）全然… ません
  • 热水与冷水
    熱いお湯　表示热水
    冷たい水　表示凉水

# 第十课　京都の紅葉は有名です

  • 二类形容词，形容动词（以だ为结尾）
    与动词肯否定，时态用法一样
    ・京都の紅葉は有名です。
    简体形式：ですーだ　ではありませんーじゃない
    でしたーだった　ではありませんでしたーじゃなかった
  • 二类形容词 + な + 名词＋です
    桜はきれいな花です。
    奈良は静かなではありません。
  • どんな + 名词
    横浜はどんな町ですか
    ーーとでもにぎやかな町です。
  • 何の和どんな
    有询问内容和材料两种用法。如：
    何の本ですかーー自動車の本です。（询问内容）
    何の料理ですかーー魚の料理です。（询问材料）
    どんな只用于询问性质。如：
    どんな本ですかーーおもしろいほんです。
    どんな料理ですかーーちよっと辛い料理です。
  • どうですか（现在）　どうでしたか（过去）
    この料理はどうですか。（这个菜怎么样？）
  • 常见的二类形容词
    好き（喜欢）　嫌い（讨厌）　にぎやか（热闹）　静か（安静）　便利（方便）　不便（不方便）　きれい（漂亮）　暇（空闲）　元気（健康）　簡単（简单）　親切（热情）　有名（有名）
  • 转移划题时使用　ところで
    でも用于口语，转折关系
    そして表示并列关系
  • 中的用法
    【场所】+ 中 世界中（全世界） 日本中（全日本） 学校中（全校）
    【时间】+ 中 一日中（一整天） 一晚中（一个晚上）

# 第十一课　小野さんは歌が好きです

  • 【名１】は【名２】が　一类形 / 二类形　です
    【名１】表示情感的主体（一般指人），【名２】表示情感的对象（一般指事）
    ・小野さんは歌が好きです。
  • 【名１】は【名２】が分かります / できます。（后者表示一件事的目的）
    表示能力的对象也可以用が（が表示情感能力的对象）
    ・スミスさんは韓国語がわかります。
  • 名や名
    "と" 用于列举所有项目，“や” 用于多项目中的两项。
    ・私はお店で野菜や果物などをかいました。
  • ～から　与　だから
    から　说明原因、理由，位于陈述原因、理由小句的句尾
    ・森さんはお酒が好きですから、毎日飲みます。
    だから　表示原因、理由的连词
    ・明日は休みです。ですから、子供と動物園へ行きます。
  • で
    表示在某处发生什么事，用 "で" 表示
  • 気に入りました
    仅用于第一次见到某事感到中意，对象用 “が” 表示，用于熟人时间，长辈不用。

# 第十二课　李さんは森さんより若いです

  • 「名 1」は「名２」より　一、二类形　です
    李さんは森さんより若いです。小李比森先生年轻
    程度差异大用 “ずっと” 强调
    中国は日本よりずっと広いです。中国比日本辽阔得多
  • 「名１」より「名２」のほうが　一、二类形　です。
    日本より中国のほうが広いです。与日本相比，中国更辽阔
  • 「名１」より「名２」ほど　一类形　くないです。<br />「名１」は「名２」ほど　二类形　ではありません。
    東京の冬は北京の冬ほど寒くないです。东京的冬天没背景的冬天那么冷
  • 「名１」（の中）で「名２」が一番　一、二类形　です
    日本で一番高い山は富士山です。日本最高的山是富士山
  • 「名１」と「名２」とどちらが　一、二类形　ですか
    日本語と英語とどちらが難しですか。
    日本語のほうが難しです。
  • どの【名】/ いつ / どれ / だれ / 何が /　一番 一类形 / 二类形　ですか
    この中でどの料理が一番おいしいですか。
  • 表示频率的词
    いつも 总是　とく 经常　時と 有时　たまに 偶尔【肯定】
    めったに 极少　ほとんど 几乎　全然 完全不【否定】
  • 为什么的三种说法
    1. どうしてですか
    2. なぜですか
    3. なんで

# 第十三课　机の上に本が３冊あります

  • 名【数量】+ 动
    量词因所修饰的事物种类而不同
    ・卵を 1 個たべます。
    吃了一个鸡蛋
  • 名【时间】+ 动
    表示数量词与动词一起使用，说明动作状态的持续时间。不加 “に”
    李さんは毎日７時間働きます。
    昼１時間休みます。
  • 名【时间】に名【次数】+ 动
    一定时间内进行若干次动作。
    ・李さんは１週間に 2 回プールへ行きます。
    この花は 2 年に 1 度咲きます。
  • 名【场所】へ动に 行きます / 来ます
    表示移动行为的目的时使用。“动” 去掉 “ます”
    ・午後郵便局へ荷物を出しに行きます。
    ・小野さんはプールへ泳ぎに行きます。
  • で复数在量词后加
    このケーキは 3 個で５００円です
    このケーキは１個２００円です
  • 表示大概数量
    くらい / ぐらい
    どのぐらい　かかりますか
    ・家から会社までどのぐらいかかりますか。
    ーー電車で一時間ぐらいかかります。
    ・修理はどのぐらいかかりますか。
    ーー２万円ぐらいかかります。
  • 几种动作或事项中，暂且选择一种
    とりあえず
    とりあえず先ビールを三つお願いします。

# 第十四课　昨日デパートへ行って、買い物しました

  • 动词「て」形
    例外：行きます的「て」形为行って
  • 动て动
    表示两个以上相继发生的动作
    昨日デパートへ行って、買い物しました。
    家へ帰って、宿題をします。回家，写作有人
    図書館へ行って本を借りて、家へ帰ります。去图书馆借书，回家。
  • 动てから动
    按时间相继发生
    李さんは毎晩ラジオを聞いてから寝ます。小李每晚听了收音机后睡觉。
    晩御飯食べてから出かけます。吃了午饭后出门。
  • 动てください
    请求某人做某事
    ここに住所と名前を書いてください。请在这里填写住址和姓名。
    この荷物を航空便で送ってください。用航空方式寄包裹。
  • 名【场所】动（经过、离开）
    经过的场所所用助词を表示
    このバスは駅前を通ります。这趟公共汽车经过车站一带。
    この道をまっすぐ行って、橋を渡ってくさい。顺着这条路一直走，再过那座桥
    李さんは毎朝７時に家を出します。小李没早 7 点离开家

# 第十五课　小野さんは今新聞を読んでいます

  • 动ています【正在进行】
    小野さんは今新聞を読んでいます。
    森さん今仕事をしています。
  • 动てもいいです【许可】
    家で仕事をしてもいいです。
  • 动てはいけません【禁止】
    ・飛行機の中でたばこを吸ってはいけない。　飞机上不能吸烟
    ・教室で物を食べてはいけません。教室里不让吃东西
  • 名【附着点】に动
    小野さんは公園でボートに乗りました。小野女士在公园里划了船
    李さんは電車に乗りました。小李上了电车
  • 名【目的地】に动　へ表方向 　に表目的
    李さんは病院に行きます。小李去了医院。
    李さんは家に帰りました。小李回家了。

# 第十六课　ホテルの部屋は広くて明るいです

  • 一类形　て　一类形
    形容词て形　一类形容词い变くて
    ホテルの部屋は広いて明るいです。宾馆的房间又宽阔又明亮
    この本は安くないです。这本书又便宜又有意思
  • 二类形で二类形
    形容动词て形，去だ加で
    このパソコンの操作は簡単で便利です。这台电脑用起来既简单又方便
  • 名 1 で名 2
    スミスさんは旅行会社の社員で、営業部の部長です。
  • 动 ています结果状态
    ・森さんは車を持っています。森先生在北京有辆车
    ・私は北京にすんでいます。我住在北京。
    注意：对” 知っていますか” 的否定回答，必须使用 “知りません”
  • 小句が，小句，表转折
    ・三か月パソを勉強しましたが、まだあまりできません。学了三个月电脑，但还是不太会用。
    注意：用もう提问做否定回答，使用まだ～ていません的表达形式
    ・小野さんはもう結婚していますか
    いいえ、まだ結婚していません。

# 第十七课　私は新しい洋服が欲しいです

  • 名が欲しいです
    用于第一人称，第二人称只用于疑问句
    私はあたらしい洋服が欲しいです。
    何が欲しいですか。
  • 名を动たいです
    用于第一人称，第二人称用于疑问句
    映画を見たいです。我想看电影
    何をしたいですか。你想什么？
    何もしたくありません。什么都不想做。
  • 动ませんか　表示提议
    一緒にお茶を飲みませんか。
    少し休みませんか。
  • 动ましょう　表示体育
    ちょっと休みましょう。
  • 疑问词＋でも
    表示任何情况下事态都想相同
    李さん、何を食べたいですか。
    何でもいいです。
    いつでも電話をしてくさい。
    誰でもわかります。
  • 对上级长辈说话
    使用どうですか或～ませんか表示尊敬
    デパートへ買い物に行きます。
    李さんも一緒にどうですか。
    はい、行きたいです。
    表示 “一定”“必定” 意思，接たい、～てください、～ましょう等表示强烈愿望意志。
    お汁粉？ぜひ食べたいです。

# 第十八课　携帯電話はとても小さくなりました

  • 一类形なります。
    表示性质或状态的变化，词尾い变为く
    暖かい→暖かくなります
    小さい→小さくなります
    安い→安くなります
    だんだん暖かくなります
    携帯電話は小さなります
    パソコンは安くなりました。
  • 一类型します
    因主语意志性动作、作用引起事物发生变化，动词变化同上
    大きい➞大きくします
    冷たい➞冷たくします
    テレビの音を大きくします。（请把电视声开大点）
    ジュースを冷たいくします。（把果汁冰镇一下）
  • 二类形 / 名になります
    もう元気になりました。
    息子は医者になしました。
  • 二类形 / 名になります
    部屋をきれいしてください。
    会議室を嫌煙にします。
  • 一类形 / 二类形容 ほうがいいです
    比较两个以上事物的性质认为其中一个较高，表示自己的选择或向别人提议
    旅行の荷物は軽いほうがいいです。
    部屋は広いほうがいいです。
  • 間もなく和もうすぐ
    都是副词，表示一会儿，不久的意思
    前者比较郑重、语气生硬用于书面语
    后者多用于口语
    間もなくタイムサービスの時間です。
    もうすぐ６時になります。
  • さらに和もっと
    都表示 “更加” 前者更郑重，后者较随意
    間もなくタイムサービスの時間です
    今から７時までもっと安くなります

# 第十九课　部屋のかぎを忘れないでください

  • 动词的ない形
    カ变和サ变：くる➞こない　する➞しない
    イ段动词：い段 / え段＋る➞去掉る加上ない
    五段动词：把最后一个音变为あ段
    如是い则变为わ再加ない

  • 动ないでください
    表示否定命令
    部屋の鍵を忘れないでください
    この部屋にいらないでください
    無理をしないでください

  • 动なければなりません
    表示必须
    李さんは今日早く帰らなければなりません。
    この薬は毎日飲まなければなりません。

  • 动なくてもいいです
    表示不做某事也可以
    明日は残念しなくてもいいです。
    慌てなくてもいいですよ。

  • 名１が名２です。＝名２は名１です
    名１为新信息
    山田先生はあの方です
    あの方が山田先生です

  • 和 + 量词 + も + 肯定形式 表示数量多的说法
    何度も転びましたが、とてもおもしろいです。
    喉が渇きましたから、水を何杯ものみました。

  • 必须做 / 可以做
    なければなりません
    なくてはなりません 社会常识、事物性质

    なければいけません
    なくではいけません 个别事物

    なければだめです
    なくてはだめです 口语
    可以做某事
    てもいいです
    なくてもいいです
    なくても構（かま）いません
    なくても大丈夫です
    省略后半部分：
    学校に通わなければ＝なきゃ
    学校に通わなくては。＝なくちゃ

  • 拜年
    明けましておめでとう
    新年！おめでとう！
    といお年をおめでとう

# 第二十课　スミスさんはピアノを弾くことができます

  • 名（人）　は　动（简）ことができます
    表示能力
    スミスさんはピアノを弾くことができます。史密斯先生会弹钢琴。
    森さんは中華料理を作ることができますか。森先生会做中国菜吗。
    名　は　动（简）ことです
    内容是一种动作行为时
    ・私に趣味は切手を集めることです。我的爱好是收集邮票。
    ・王さんの特技は胡弓を弾くことです。小王的特长是拉胡琴
  • 趣味　和　興味
    前者是爱好喜好、后者是兴趣
  • 授受动词
    あげる、さしあげる、やる，我给别人
    もらう、いただく，从别人得到
    くれる、くださる，别人给我
    1. 我或我方的人给别人某东西，“给”
      やる：给比自己身份低的人
      私は魚に餌（えさ）をやりました
      あげる：给同辈人
      李さんは劉さんに本を上げます。
      さしあげる：给长辈
      私は先生に本をさしあげます。
    2. 请别人给我或我方人东西
      くれる：授予者为亲人、同辈或晚辈时
      兄さんは私にお菓子をくれました
      くださる：授予者身份高
      先生は妹にプルゼンとをくださいました。
    3. 我或我方从别人那里得到什么
      いただく：对方地位高
      もらう：地方地位低或和自己平等
      誕生日に友達からプレゼントをもらいました。
  • たがる和がっている区分
    前者表示倾向性、后者表示状态

# 第二十一课　私はすき焼きを食べたことがあります

  • 动（た形）後で、～<br /> 名词 + の＋後で、
    表示一个动作在另一个动作之后发生
    例：会社が終わった後で、飲みに行きます。
    仕事の後で、映画を見ます。（工作之后去看电影）
  • 动（た形）ことがあります
    表示过去的经历，通常用于很久前发生的事
    回答：はい、あります / いいえ、ありません
    例：私はすき焼きを食べたことがあります。（我吃过日式牛肉火锅）
    北京へ行ったことがありますか。
  • 动（た形）ほうがいいです
    用于两种事物中进行选择
    例：もっと野菜を食べたほうがいいです（多吃带你蔬菜好啊）

# 第二十二课　森さんは毎晩テレビを見る

  • 动词 “た” 形
    动词过去式是ました的简体
    链接其他语法：
    （1）动【た形】+ ことがあります　曾经做过某事
    动【た形】+ ことがありません　否定
    动【た形】+ ことがありますか　疑问
    （2）动【た形】/ 名の + 後で　表示做某事之后
    （3）动【た形】ほうがいいです　表建议
    动【ない形】ほうがいいです　最好不要做

  • 动词形式

    --敬体简体
    现代肯定買います買う
    现代否定買いません買わない
    过去肯定買いました買った
    过去否定買いませんでした買わなかった
    ある形式敬体简体
    现代肯定ありますある
    现代否定ありませんない
    过去肯定ありましたあった
    过去否定ありませんでしたなかった

  • 形容词形式

    --敬体简体
    现代肯定忙しいです忙しい
    现代否定忙しくないです忙しくない
    过去肯定忙しかったです忙しかった
    过去否定忙しくなかったです忙しくなかった

    そのカレー、おいしい
    ううん、あんまり美味しくないよ

  • 形容动词和名词形式

    ---敬体简体
    现代肯定簡単です簡単だ
    现代否定簡単ではありません簡単ではない
    过去肯定簡単でした簡単だった
    过去否定簡単ではありませんでした簡単ではなかった

    このネクタイ、派手
    ううん、全然派手じゃないわ

  • 【小句】けと，【小句】
    表示转折：
    ちょっと難しかったけど、まあまあできました。
    用来铺垫：
    来週、送別会を巣すろけど、都合はどうかな？

  • 简体助词

    1. かな
      用于自言自语的场合，让对方听到自己的自问自答而提供一种不太确信的信息
      例：今日李さんは来るかな
    2. の　用于要求说明或确定某事
      寒いの？（冷吗）

  • ~ って 表示传闻
    例：太田、今度中国へ転勤だって
    听说太田要调到中国去工作

# 第二十三课　休みの日、サンボしたり買い物に行ったりします

  • 【** 动】たり【动】たりします　**
    表示列举若干种代表性的动作
    例：小野さんは休みの日、散歩したり買い物に行ったりします。
    周末或节假日，小野女士有时去散步，有时去买东西
  • 【形】かったり【形】かったりです <br />【形动 / 名词】だったり【形动 / 名词】だったりです
    例：飛行機のチケットは高いかったり安かったりです。
  • 小句 + か
    1. + かどうか 表示某种不确定的内容
      例：わたしは今年の夏、北京へ行くかどうかわかりません。
      今年夏天去不去北京，还不知道。
    2. 疑问词小句 + か
      例：鍵がどこにあるか教えてください。请告诉我钥匙在哪儿。
  • 区分：（肯定，一定）
    1. きっと（非常有把握的判断）
      一定会去：きっといきます。
    2. ぜひ　（愿望）一定
      一定想要：ぜひ欲しい
    3. 必ず　义务、要求

# 第二十四课　李さんはもうすぐ来ると思います

  • 小句（简体）とおもいます
    表示思考的内容
    李さんはもうすぐ来ると思います。我想小李马上就来。
    第三人称
    馬さんは日本へ行きたいと思っています。（小马想去日本）
  • 名【人】小句（简）と言いました
    向第三者转述他人说的话
    陳さんはパーティーに行くと言いました。（老陈说要去参加联谊会）
    转述反复说起的事情
    来週のハイキングに張さんも行きたいと言っています。

# 总结

  • 长音
    あ、い、う、え、お段
    对应加あ、い、う、い、う变为长音
    特殊：大きい、姉さん
  • 汉字词的规律
    音读
    当日文汉字词对应中文有该意思，则类似于中文音
    训读
    当日文中单独一个汉字有意思，则与中文音没有关联
  • 动词
    动词分类
    カ变
    来る
    サ变
    ～する
    一段
    ～い / う段 + る
    五段
    う、く、す、つ、ぬ、ぶ、む、る、ぐ
    凡是不以る结尾的动词都是五段动词
    动词变化
    カ变
    来 (ku) るー＞来（ki）
    サ变
    ～する
    する —> し
    一段
    る前的一个假名在い段或え段上
    去掉る
    五段
    一型，ます型
    往前推一段
    う、く、す、つ、ぬ、ぶ、む、る、ぐ
    变为い、き、し、ち、に、び、み、り、ぎ
    二型，た型
    促音型う、つ、る
    例：打つ
    加て变为，打って变为促音
    イ音型
    く、す、ぐ变为い、し、（い（后加浊音））
    拨音型
    ぬ、ぶ、む
    变为ん，然后加浊音
    第一连用型
    加ます、ません、ました、ませんでした
    第二连用型
    单词
    た表示过去（简体）
    て 连续
    てから 相继
    てください 请
    てもいいです 允许，可以
    てはいけません 不可以
    用法
    + て～
    走って来ます
  • 助词
    は主题
    が主语
    传递信息重点
    自然现象

は（大）が（小）

  • 单词变化
    促音变化
    【か、さ、た、は行】+【か、さ、た行】
    【か、さ、た、は行】变促音
    【か、さ、た、は行】+【は行】
    【か、さ、た、は行】变促音，后は变为ぱ
    学生、学校
    拨音变
    【ん】+【か、さ、た、は行】
    【か、さ、た、は行】变浊音
  • 量词
    ～人（nin）
    一人ひとり
    二人ふたり
    三人以上　　～人（にん）
    ～つ
    不能用于形容 10 以上的数
    帽子，铅笔，选项等
    单独记忆
    ～本（hon）
    细长的物品：瓶、铅笔、伞、香蕉
    ～台（dai）
    电器
    车辆、手机等
    ～匹（hiki）
    小动物
    ～尾（bi）
    鱼
    ～頭（tou
    大动物
    ～個 (ko)
    能握在手里的
    特殊：三個（sanko）不变波音
