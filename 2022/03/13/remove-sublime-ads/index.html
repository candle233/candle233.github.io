# sublime 下载地址

  1. 如果打开了 Sublime text，就将其关闭（为接下来备份做准备）
  2. 打开 sublime text 的安装目录，然后备份一下 sublime_text.exe（防止失败需要重装软件）
  3. 打开网址：https://hexed.it
    * 这个网址仅仅是一个在线的 16 进制编辑器，可以修改程序，如果打不开网址 / 网速过慢，可以用本地的 16 进制编辑器例如 hxd 编辑器等代替
  4. 点击上面的 Open File，弹出的对话框中选择 sublime_text.exe。
  5. Ctrl+F 搜索查找 ，输入 97 94 0D，然后点击按钮 Search Now。
  6. 在 97 94 0D 上面依次点击，将其替换为 00 00 00
  7. 点击上面的 Export 按钮导出，将完成后的文件复制到刚才的目录替换掉原来的文件即可
  8. 打开 Sublime text，然后点击菜单 Help->Enter Lisence:（这只是随便的一个，是我测试成功的）全部都要复制，包括 BEGIN 和 END

# 另一种方式

在 C:\Windows\System32\drivers\etc 路径，添加 host 屏蔽

127.0.0.1 activate.navicat.com
127.0.0.1 license.sublimehq.com
127.0.0.1 45.55.255.55
127.0.0.1 45.55.41.223
然后输入激活码即可
