# JAVA

下载相应版本的 JDK：JDK8 下载链接
备用下载链接
下载 jdk 相应的版本，安装并记住目录。
打开高级系统设置（可以通过 Windows 的搜索功能即 Win+S），然后找到环境变量打开，检验是否 JAVA 已经完全删除。然后新建，输入 JAVA_HOME 和路径，然后点击确定。


再系统变量内找到 Path，双击打开。创建红色标记的两项然后确定并退出。

最后输入 java -version 确认 java 的版本
